\subsection{Detailed Description}
Display Resolutions. A list of (theoretically) all the possible display resolutions, ordered by width, then height. Taken from \href{https://upload.wikimedia.org/wikipedia/commons/0/0c/Vector_Video_Standards8.svg}{\tt https\-://upload.\-wikimedia.\-org/wikipedia/commons/0/0c/\-Vector\-\_\-\-Video\-\_\-\-Standards8.\-svg} and Steam's 2015 Hardware Survey. Last updated on July 2015.
